# SOC Brute Force Detection Lab

## Overview
This project demonstrates how to detect and investigate brute-force login activity in a SOC-style lab environment.

## Objective
The goal is to identify repeated failed login attempts and analyze them as a possible brute-force attack.

## Tools
- Windows VM
- Ubuntu VM
- Wazuh
- VS Code
- VirtualBox

## Use Case
Detect multiple failed login attempts within a short time.

## Status
Lab setup in progress.
